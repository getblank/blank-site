Снижение затрат на разработку

У нас готов пользовательский интерфейс, мы решили множество рутинных проблем обработки данных. Осталось запрограммировать логику работы вашей компании.

Развитие вместе с компанией

Blank максимально расширяем, и будет легко расти и меняться вместе с вами. Кроме того, наша команда каждый день делает Blank лучше и добавляет в него новые возможности

Экономия времени

Мы поможем обеспечить кратчайшие сроки внедрения, предоставим поддержку и услуги по разработке

Технологии

Blank разработан на современных и широко известных языках программирования — Go и JavaScript.
Клиентская часть — SPA на платформе React JS. Данные в приложении обновляются в режиме реального времени благодаря двунаправленному каналу связи с сервером по протоколу WebSocket.
При разработке на Blank не нужно изучать Go — все делается на JavaScript, для этого обработчики задач реализованы на Node JS
Подробнее об API для разработчиков читайте в документации
